# PalindromeCheckerApp

A simple Java application that determines whether a given word, number, or phrase is a palindrome — a sequence that reads the same forwards and backwards (ignoring punctuation and case).

## 🧠 What This Project Does

**PalindromeCheckerApp** is a utility that lets users input text and checks if it’s a palindrome.

A palindrome is defined as a string that **reads the same in reverse as forward**, e.g., `"racecar"`, `"madam"`, or `"A man, a plan, a canal: Panama"` once normalized.
